## 📋 Requirements

### System Requirements
- **Python 3.13+** (Optimized for asynchronous execution)

- **OpenAI API Key** (Cloud Core) & **Ollama** (Local Core).
- **Docker & Kubernetes** (For distributed cloud deployment).
